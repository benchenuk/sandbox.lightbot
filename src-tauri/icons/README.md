# Icons

Place your app icons here:

- `icon.png` - 512x512 main icon (used for tray)
- `32x32.png` - 32x32 icon
- `128x128.png` - 128x128 icon
- `128x128@2x.png` - 256x256 icon (Retina)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon bundle

You can generate these from a single SVG using the Tauri icon command:
```bash
npm run tauri icon /path/to/icon.svg
```
